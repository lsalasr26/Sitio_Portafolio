ProJob

ProJob fue el trabajo final del curso Proyecto de Ingeniería de Software 1 del plan de estudio de Bachillerato en Ingeniería de Software, primer cuatrimestre de la Universidad CENFOTECT. El proyecto consistía en crear una página de empleos como iniciativa del gobierno de Costa Rica. Las tecnologías utilizadas fueron HTML, CSS, JavaScript, MongoDB Atlas y Node.js. También se utilizaron FontAwesome y SweetAlert.

Pet's Inn

Pet's Inn es el proyecto final del curso Proyecto de Ingeniería de Software 2 del tercer cuatrimestre del plan de estudios de Bachillerato en Ingeniería de Software de la Universidad CENFOTECT. Consiste en un hotel para mascotas que ofrece la posibilidad de comprar servicios y realizar pagos mediante PayPal. Este proyecto fue desarrollado utilizando HTML, CSS, JavaScript, Bootstrap, una API RESTful y .NET, y fue desplegado a través de Azure.

Caroline Sada

El proyecto Caroline Sada es parte del programa Software Developer Core en la modalidad Fast Track ofrecido por Digital Nao Tecmilenio en México. Este programa está diseñado para certificar las competencias de profesionales capaces de diseñar, programar, construir, desplegar y mantener software. Incluye supervisión y participación directa en procesos de programación, diseño, prueba y mejora de software. Este proyecto corresponde al segundo reto de Diseño Responsivo Front End en JavaScript, utilizando HTML, CSS, JavaScript, Bootstrap y templates.
