ProJob

ProJob was the final project of the Software Engineering Project 1 course from the Bachelor's Degree program in Software Engineering, first semester at Universidad CENFOTECT. The project involved creating a job listing page as an initiative by the government of Costa Rica. The technologies used were HTML, CSS, JavaScript, MongoDB Atlas, and Node.js. FontAwesome and SweetAlert were also utilized.

Pet's Inn

Pet's Inn is the final project of the Software Engineering Project 2 course in the third semester of the Bachelor's Degree program in Software Engineering at Universidad CENFOTECT. It consists of a pet hotel that offers the possibility to purchase services and make payments via PayPal. This project was developed using HTML, CSS, JavaScript, Bootstrap, a RESTful API, and .NET, and was deployed through Azure.

Caroline Sada

The Caroline Sada project is part of the Software Developer Core program in the Fast Track modality offered by Digital Nao Tecmilenio in Mexico. This program is designed to certify the competencies of professionals capable of designing, programming, building, deploying, and maintaining software. It includes supervision and direct participation in programming, design, testing, and software improvement processes. This project corresponds to the second challenge of Responsive Front End Design in JavaScript, using HTML, CSS, JavaScript, Bootstrap, and templates.
